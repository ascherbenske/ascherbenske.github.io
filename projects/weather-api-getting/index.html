End-to-End Weather Data

2026 · python · APIs · BeautifulSoup · pandas

This project demonstrates the techniques required to build an end-to-end data pipeline to exercise my ability to do some basic data gathering, transformation, and visualization. It pulls the top five to six cities in each state from a Wikipedia page, latitude and longitude from Open Weather Map, and weather data from Open Meteo. The pipeline cleans and merges using pandas and then visualizes the data with Plotly.
